## Purpose

This folder is the **Drop Zone** for the user (Manager).

## How to use

1. Create a file here with **ANY** content:
   - A raw paste of a cool config you found online.
   - A snippet of an error log you don't understand.
   - A brain dump of features you want.
2. Tell Antigravity: "Process the inbox."

## Antigravity's Job

- Read files in this folder.
- Refactor them into proper Nix modules.
- Update `PLAN.md`.
- Delete the file from Inbox once processed.
